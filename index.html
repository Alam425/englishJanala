Frequently Asked Questions

What is the difference between `var`, `let`, and `const`.
All are used to declare any variables. But, `var` can hoist any reference variables below itself. `let` and `const don't hoist, but variables declared with `const` can't be changed.
What is the difference between `map()`, `forEach()`, and `filter()`?
All are used to loop through any array. `map()` method is used to loop an array and get all the elements in it. `forEach()` method is used to loop an array too, but it doesn't return anything. `filter()` method is used to loop and array and get only those elements that match the given condition.
Explain arrow functions and how they are different from regular functions.
For writing regular functions, we use the `function` keyword, then the function name, then the arguments, then we start to write all the functionalities. But, in arrow function, we don't need to use the ` function` keyword, just write the function name as the variable name, then put the arguments, and then we can start to write all the functionalities after `=>`. Sometimes, we can write a function within a single line with arrow function.
How JavaScript Promises work?
There is a method, called `fetch`. We can put a api url to it. Then, JavaScript promises to provide us the api. It is asynchronous. So, how even long the api responses, it will push itself to the queue. If, it succeeds, it show the api data. If it fails, then it says through `catch()` method.
How closures work in JavaScript?
Closure is a nested function, that can capture the variables from it's parent function or outer scope.
